üñ®Ô∏è The Print Statement

The print() function is your program's voice! It displays text, numbers, and results on the screen.

üí° Pro Tip

Use quotes for text, no quotes for numbers. You can print multiple things by separating them with commas!

Example Code:
print("Hello, World!")
print("Welcome to Python programming!")
print(2024) # Numbers don't need quotes
print("The year is", 2024) # Mix text and numbers

üì¶ Variables - Your Data Containers

Variables are like labeled boxes that store information. You can put data in them, change the contents, and use them throughout your program!

Variable Magic:
name = "Alex" # Text (string)
age = 16 # Number (integer)
height = 5.6 # Decimal (float)
is_student = True # True/False (boolean)

# Use variables in print statements
print("My name is", name)
print("I am", age, "years old")

üó£Ô∏è User Input - Making Programs Interactive

The input() function lets your program ask questions and get answers from users. This makes your programs interactive and personal!

Interactive Example:
