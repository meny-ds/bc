ğŸ¯ Accessing List Elements

Each item in a list has a position called an "index". Python starts counting from 0, so the first item is at index 0, the second at index 1, and so on!

apple
banana
cherry
date
elderberry

ğŸ“ Positive Indexing

fruits = ["apple", "banana", "cherry", "date"]

print("First fruit:", fruits[0]) # apple
print("Second fruit:", fruits[1]) # banana
print("Third fruit:", fruits[2]) # cherry
print("Fourth fruit:", fruits[3]) # date

# Get multiple elements
print("First two:", fruits[0:2]) # ['apple', 'banana']
print("Last two:", fruits[2:4]) # ['cherry', 'date']

ğŸ”„ Negative Indexing

fruits = ["apple", "banana", "cherry", "date"]

print("Last fruit:", fruits[-1]) # date
print("Second last:", fruits[-2]) # cherry
print("Third last:", fruits[-3]) # banana
print("Fourth last:", fruits[-4]) # apple

# Slicing with negative indices
print("Last two:", fruits[-2:]) # ['cherry', 'date']
print("All except last:", fruits[:-1]) # ['apple', 'banana', 'cherry']

ğŸ¯ Indexing Rules

â€¢ **Index starts at 0**: First item is [0], not [1]
â€¢ **Negative indices**: Count backwards from the end
â€¢ **Slicing [start:end]**: Get multiple items
â€¢ **Out of bounds**: Accessing non-existent index causes error

ğŸ”„ List Operations and Methods

Lists are dynamic! You can add, remove, and modify items. This makes them perfect for managing changing data like todo lists, game inventories, or user collections.

â• Adding Items

# Add items one by one

â– Removing Items

# Remove last item
last_item = shopping_cart.pop()
print(f"Last removed: {last_item}")
print("Final cart:", shopping_cart)

ğŸ” Accessing and Checking

game_stats = {
    "player": "Alex",

ğŸ”„ Iterating Through Data

scores = {
    "Alice": 95,
    "Bob": 87,
    "Charlie": 92,
    "Diana": 98
}

# Loop through keys
print("Students:")
for name in scores:
    print(f"- {name}")

# Loop through key-value pairs
print("Scores:")
for name, score in scores.items():
    print(f"{name}: {score}%")

# Find highest score
best_student = max(scores, key=scores.get)
print(f"Top student: {best_student} ({scores[best_student]}%)")

â€¢ **del dict[key]**: Remove key-value pair
â€¢ **dict.pop(key)**: Remove and return value
â€¢ **dict.keys()**, **dict.values()**, **dict.items()**: Get collections
