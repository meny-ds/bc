📋 Introduction to Lists

Lists are like containers that can hold multiple items in order. Think of them as a shopping list, a playlist, or a collection of your favorite things!

🛒 Shopping List Example

# Creating a shopping list
shopping_list = ["apples", "bread", "milk", "eggs"]

print("My shopping list:")
for item in shopping_list:
    print(f"- {item}")

# List length
print(f"I need to buy {len(shopping_list)} items")

🎵 Music Playlist

# Different types in one list
playlist = ["Bohemian Rhapsody", "Imagine", "Hotel California"]
ratings = [5, 4, 5, 3, 4]
mixed_data = ["Song", 2024, True, 4.5]

print("Favorite songs:", playlist)
print("Ratings:", ratings)
print("Mixed data:", mixed_data)

🔑 List Essentials

• **Square brackets []**: Create lists
• **Commas**: Separate items
• **Any data type**: Numbers, strings, booleans
• **len()**: Get the number of items
• **Ordered**: Items stay in the order you put them

🎯 Accessing List Elements

Each item in a list has a position called an "index". Python starts counting from 0, so the first item is at index 0, the second at index 1, and so on!

apple
banana
cherry
date
elderberry

📍 Positive Indexing

fruits = ["apple", "banana", "cherry", "date"]

print("First fruit:", fruits[0]) # apple
print("Second fruit:", fruits[1]) # banana
print("Third fruit:", fruits[2]) # cherry
print("Fourth fruit:", fruits[3]) # date

# Get multiple elements
print("First two:", fruits[0:2]) # ['apple', 'banana']
print("Last two:", fruits[2:4]) # ['cherry', 'date']

🔄 Negative Indexing

fruits = ["apple", "banana", "cherry", "date"]

print("Last fruit:", fruits[-1]) # date
print("Second last:", fruits[-2]) # cherry
print("Third last:", fruits[-3]) # banana
print("Fourth last:", fruits[-4]) # apple

# Slicing with negative indices
print("Last two:", fruits[-2:]) # ['cherry', 'date']
print("All except last:", fruits[:-1]) # ['apple', 'banana', 'cherry']

🎯 Indexing Rules

• **Index starts at 0**: First item is [0], not [1]
• **Negative indices**: Count backwards from the end
• **Slicing [start:end]**: Get multiple items
• **Out of bounds**: Accessing non-existent index causes error

🔄 List Operations and Methods

Lists are dynamic! You can add, remove, and modify items. This makes them perfect for managing changing data like todo lists, game inventories, or user collections.

➕ Adding Items

# Start with an empty list
todo_list = []

# Add items one by one
todo_list.append("Study Python")
todo_list.append("Walk the dog")
todo_list.append("Buy groceries")

print("Todo list:", todo_list)

# Insert at specific position
todo_list.insert(1, "Call mom")
print("Updated list:", todo_list)

# Add multiple items
todo_list.extend(["Exercise", "Read book"])
print("Final list:", todo_list)

➖ Removing Items

shopping_cart = ["laptop", "mouse", "keyboard", "monitor"]

print("Cart:", shopping_cart)

# Remove specific item
shopping_cart.remove("mouse")
print("After removing mouse:", shopping_cart)

# Remove by index
removed_item = shopping_cart.pop(1)
print(f"Removed: {removed_item}")
print("Cart now:", shopping_cart)

# Remove last item
last_item = shopping_cart.pop()
print(f"Last removed: {last_item}")
print("Final cart:", shopping_cart)

🔍 Finding and Counting

numbers = [1, 3, 7, 3, 9, 3, 5]

# Find index of item
print("Index of 7:", numbers.index(7))

# Count occurrences
print("Count of 3:", numbers.count(3))

# Check if item exists
print("Is 9 in list?", 9 in numbers)
print("Is 10 in list?", 10 in numbers)

# Sort the list
numbers.sort()
print("Sorted:", numbers)

# Reverse the list
numbers.reverse()
print("Reversed:", numbers)

🔄 Modifying Items

grades = [85, 90, 78, 92, 88]

print("Original grades:", grades)

# Change specific grade
grades[2] = 82 # Improve the 78
print("After improvement:", grades)

# Change multiple grades
grades[0:2] = [87, 93]
print("After updates:", grades)

# Calculate average
average = sum(grades) / len(grades)
print(f"Average grade: {average:.1f}")

🛠️ Essential List Methods

• **append()**: Add item to end
• **insert(index, item)**: Add item at position
• **remove(item)**: Remove first occurrence
• **pop(index)**: Remove and return item
• **sort()**: Sort the list
• **reverse()**: Reverse the order

🗂️ Introduction to Dictionaries

Dictionaries store data in key-value pairs, like a real dictionary where you look up a word (key) to find its meaning (value). They're perfect for organizing related information!

👤 Person Profile

# Creating a person dictionary
person = {
    "name": "Alex",
    "age": 16,
    "city": "New York",
    "student": True,
    "grade": 10
}

print("Person info:", person)
print("Name:", person["name"])
print("Age:", person["age"])
print("Is student:", person["student"])

🏪 Product Information

# Product in an online store
product = {
    "id": "LAP001",
    "name": "Gaming Laptop",
    "price": 1299.99,
    "in_stock": True,
    "quantity": 5,
    "category": "Electronics"
}

print(f"Product: {product['name']}")
print(f"Price: ${product['price']}")
print(f"Available: {product['quantity']} units")

🔑 Dictionary Essentials

• **Curly braces {}**: Create dictionaries
• **Key-value pairs**: "key": value
• **Unique keys**: Each key appears only once
• **Any data type**: Values can be strings, numbers, booleans, lists
• **Unordered**: Items don't have a fixed position

🔑 Working with Keys and Values

Dictionaries are incredibly flexible! You can add new key-value pairs, update existing ones, remove items, and search through your data efficiently.

➕ Adding and Updating

# Start with basic info
student = {
    "name": "Emma",
    "grade": 11
}

print("Original:", student)

# Add new information
student["age"] = 16
student["subjects"] = ["Math", "Science", "English"]
student["gpa"] = 3.8

print("After adding:", student)

# Update existing information
student["grade"] = 12
student["gpa"] = 3.9

print("After updating:", student)

🔍 Accessing and Checking

game_stats = {
    "player": "Alex",
    "score": 1500,
    "level": 8,
    "lives": 3
}

# Safe access with get()
print("Player:", game_stats.get("player"))
print("Score:", game_stats.get("score", 0))
print("Power:", game_stats.get("power", "None"))

# Check if key exists
if "lives" in game_stats:
    print(f"Lives remaining: {game_stats['lives']}")

# Get all keys and values
print("All keys:", list(game_stats.keys()))
print("All values:", list(game_stats.values()))

➖ Removing Items

inventory = {
    "sword": 1,
    "shield": 1,
    "potion": 5,
    "key": 2,
    "junk": 10
}

print("Before cleanup:", inventory)

# Remove specific item
del inventory["junk"]
print("After removing junk:", inventory)

# Remove and get value
used_potions = inventory.pop("potion", 0)
print(f"Used {used_potions} potions")
print("Final inventory:", inventory)

🔄 Iterating Through Data

scores = {
    "Alice": 95,
    "Bob": 87,
    "Charlie": 92,
    "Diana": 98
}

# Loop through keys
print("Students:")
for name in scores:
    print(f"- {name}")

# Loop through key-value pairs
print("Scores:")
for name, score in scores.items():
    print(f"{name}: {score}%")

# Find highest score
best_student = max(scores, key=scores.get)
print(f"Top student: {best_student} ({scores[best_student]}%)")

🛠️ Essential Dictionary Methods

• **dict[key] = value**: Add or update
• **dict.get(key, default)**: Safe access
• **key in dict**: Check if key exists
• **del dict[key]**: Remove key-value pair
• **dict.pop(key)**: Remove and return value
• **dict.keys()**, **dict.values()**, **dict.items()**: Get collections
