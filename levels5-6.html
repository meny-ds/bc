üî¢ For Loops - Counting and Iterating

For loops are perfect when you know exactly how many times you want to repeat something. They're like having a super-efficient assistant that can count and repeat tasks!

üéØ Basic Counting

# Count from 1 to 5
for i in range(1, 6):
    print(f"Count: {i}")

# Count backwards
for i in range(5, 0, -1):
    print(f"Countdown: {i}")

print("Blast off! üöÄ")

üåü Patterns and Shapes

    stars = "‚≠ê" * i
    print(stars)

# Multiplication table

for i in range(1, 11):
    result = number * i
    print(f"{number} √ó {i} = {result}")

üîë For Loop Essentials

‚Ä¢ **range(n)**: Counts from 0 to n-1
‚Ä¢ **range(start, stop)**: Counts from start to stop-1
‚Ä¢ **range(start, stop, step)**: Counts with custom steps
‚Ä¢ **Indentation**: Everything inside the loop must be indented

‚è∞ While Loops - Conditional Repetition

While loops continue running as long as a condition is true. They're perfect for situations where you don't know exactly how many iterations you need!

üé≤ Number Guessing Game

import random

secret = random.randint(1, 10)
guess = 0
attempts = 0

print("üéØ Guess the number (1-10)!")

while guess != secret:
    guess = int(input("Your guess: "))
    attempts += 1

    if guess < secret:
        print("üìà Too low!")
    elif guess > secret:
        print("üìâ Too high!")
    else:
        print(f"üéâ Correct! It took {attempts} attempts!")

# Shopping list with prices

‚ö†Ô∏è While Loop Safety

‚Ä¢ **Always update the condition**: Make sure the loop variable changes
‚Ä¢ **Avoid infinite loops**: Ensure the condition will eventually become false
‚Ä¢ **Use break**: Exit loops early when needed
‚Ä¢ **Use continue**: Skip to the next iteration

üîß Functions - Reusable Code Blocks

Functions are like mini-programs within your program. They help you organize code, avoid repetition, and create modular, maintainable applications.

üéØ Basic Functions

# Define a function
def greet(name):
    return f"Hello, {name}! üëã"

# Call the function
message = greet("Alice")
print(message)

# Function with multiple parameters
def calculate_area(length, width):
    area = length * width
    return area

result = calculate_area(10, 5)
print(f"Area: {result} square units")

üßÆ Advanced Functions

# Function with default parameters

    profile = {
        "name": name,
        "age": age,
        "city": city
    }
    return profile

# Call with different parameters

üîë Function Essentials

‚Ä¢ **def**: Keyword to define a function
‚Ä¢ **Parameters**: Input values the function accepts
‚Ä¢ **return**: Send a value back from the function
‚Ä¢ **Scope**: Variables inside functions are local

üìö Dictionaries and Data

# Student database
students = [
    {"name": "Alice", "age": 20, "grades": [85, 92, 78]},
    {"name": "Bob", "age": 19, "grades": [88, 84, 91]},
    {"name": "Charlie", "age": 21, "grades": [92, 89, 94]}
]

# Calculate averages
for student in students:
    avg = sum(student["grades"]) / len(student["grades"])
    student["average"] = avg
    print(f"{student['name']}: {avg:.1f}")

# Find top student
top_student = max(students, key=lambda s: s["average"])
print(f"Top student: {top_student['name']}")
